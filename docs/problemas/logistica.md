# Logistica de campo

## Dor

Deslocamentos longos, equipes distribuidas e suprimentos exigem organizacao que
nao existe de forma padronizada.

## Impacto

- Custos maiores.
- Retrabalho e falhas de cobertura.
- Dificuldade para planejar viagens.

## Direcao de solucao

- Checklists simples para viagens e eventos.
- Calendario basico de atividades.
- Controle simples de estoque e necessidade.
